## :fire: Installation
By default, this project uses [NPM](https://www.npmjs.com) as depedency/package management.
1. Install dependencies:
```
npm install
```
2. Run:
   - 2.1. Development (hot-reloads):
   ```
   npm run dev
   ```
   - 2.2. Production (compiles and minifies):
     - 2.2. Build:
     ```
     yarn run build
     ```
     - 2.2. Start:
     ```
     yarn run start
     ```
